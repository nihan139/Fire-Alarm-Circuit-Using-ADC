# Fire-Alarm-Circuit-Using-ADC
Fire Alarm Circuit using custom 4 bit Analog to Digital Converter

## **Components**
1. LM35   (Temp Sensor)
2. LM324 (Quad Op-Amp)
3. IC7805 (5V REGULATOR)
4. IC7404 (NOT)
5. IC7408 (AND)
6. IC7432 (OR)
7. IC7486 (XOR)
8. LED, BUZZER, POT
9. 9V Battery
10. BREADBOARD
11. JUMPER
